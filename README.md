### Why NeuroCache?

Traditional cache algorithms (LRU, LFU, ARC) treat data as static.  
NeuroCache introduces **memory decay, reinforcement, and consolidation**, inspired by how the human brain learns.  
This allows systems to **adapt automatically** to changing data patterns and workloads.
